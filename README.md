### Run Locally

To run **Tourest** locally, run this command on your git bash:

Linux and macOS:

```bash
sudo git clone https://github.com/YoussefYousry994/tourest.git
```

Windows:

```bash
git clone https://github.com/YoussefYousry994/tourest.git
```
